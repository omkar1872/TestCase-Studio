# TestCase-Studio

**TestCase-Studio** is a Streamlit web app that generates clean software test cases from requirement documents using Groq LLM.

---

## Features
- Upload TXT, PDF, or DOCX requirement files **or** paste text directly.
- Generate detailed test cases in a clean, structured format.
- Download generated test cases as a TXT file.
- Modern, user-friendly interface with light theme.
